## Adjustment Triggers

Plan flexes on these signals — re-baseline if hit:

| Trigger | Response |
|---|---|
| LTHR retest shifts >5 bpm | Regenerate all zones, re-anchor HM pace target |
| Illness ≥ 5 days | Drop back 2 weeks in calendar; volume at 70% for 1-2 weeks |
| Shin/calf flare | Cut volume 30%, hold quality, add strength frequency |
| CTL drops >5 pts vs plan after recovery week | Was overload — extend recovery, lower next block ceiling |
| Decoupling >5% on long runs for 2 consecutive weeks | Hold volume, drop intensity 5-7 days |
| HRV 7-day avg < baseline_low for 4+ days | Same as accumulated fatigue protocol (Decision Matrix #4) |
