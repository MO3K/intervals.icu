# Strength Program — Injury-Prone Runner

**Athlete context:** Volodymyr, calf/shin splints history, runs 4-5×/week + 2× cycling.
**Goal:** Injury prevention + running economy + power-to-weight for HM goal.
**Frequency:** 1 mandatory + 1 optional session per week.

---

## Session A — Monday: Trainer cycling (AM) + Gym (post-meal)

**Why Monday:** Far from Wed Long Run (>48h) and Sat quality (>5 days). Combining the trainer session and gym on the same day stacks all aerobic + strength load away from running quality sessions.

### Sequence

**1. Morning: Trainer cycling Z2 (50–60 min, ERG mode)** — the standard Mon cycling block, see week plan for power targets.

**2. Eat + recover (~1 hour).** Carbs + protein. Prevents fasted strength.

**3. Gym Block A1 — Upper Body (30–40 min):**

| Exercise | Sets × Reps | Notes |
|---|---|---|
| Pull-ups (assisted if needed) or lat pulldown | 3×6–10 | Builds posture-supporting back |
| DB bench press or push-ups | 3×8–12 | Push counterbalance |
| One-arm DB row | 3×8–10 per side | Unilateral, core stability |
| DB shoulder press | 3×8–10 | Overhead strength, prevents rounded shoulders |
| Face pulls or band pull-aparts | 3×12–15 | Rear delts, fixes desk posture |
| Plank (front + side) | 3×30–45s | Core anti-extension/lateral |

**3. Gym Block A2 — Leg Prevention (15–20 min):**

| Exercise | Sets × Reps | Why |
|---|---|---|
| **Eccentric calf raises** (3s down) on step | 3×12 straight knee + 3×12 bent knee | Soleus + gastroc, Alfredson protocol — gold standard for shin/calf prevention |
| **Tibialis raises** (toes up against wall or band) | 3×15–20 | Anterior shin — directly prevents shin splints |
| **Single-leg glute bridges** | 3×10 per side | Glute med, hip stability |
| **Single-leg deadlift** (BW or light DB) | 3×8 per side | Posterior chain, balance, ankle stability |

**Never to failure** on leg work — these are durability sessions, not strength PRs.

**Total time:** Trainer 50–60 min + eat 60 min + gym 45–60 min = whole morning/afternoon block.

**Load progression (upper):** Add 2.5 kg when last set hits top of rep range with 2 reps in reserve.
**Load progression (legs):** Weeks 1–4 bodyweight + slow 3s eccentric. Weeks 5–8 add 5 kg DB on calf raises and SL deadlifts.

---

## Session B — Optional Thursday: Bodyweight at home (15–20 min)

**Why optional and why Thursday only:**

Constraint: must NOT precede a quality/test session.
- Mon — Session A already (skip)
- Tue — precedes Wed Long Run → AVOID
- Wed — same as Long Run → AVOID
- **Thu — recovery day, far from any quality → OK ✓**
- Fri — precedes Sat Tempo or test → AVOID
- Sat — same as Tempo/test → AVOID
- Sun — REST day → AVOID

So **Thursday is the only valid slot**. Skip if time short or feeling cumulative fatigue.

### Focus: core + mobility, NOT additional leg strength

(Legs already worked Mon. Thursday session is upkeep, not loading.)

| Exercise | Sets × Reps | Why |
|---|---|---|
| Dead bug | 3×10 per side | Core anti-rotation |
| Bird dog | 3×10 per side | Posterior chain coordination |
| Side plank with leg lift | 2×10s each side | Hip abductor + lateral core |
| Wall sit | 1×45–60s | Quad isometric (light, recovery-compatible) |
| Pigeon stretch | 1×60s per side | Hip mobility |
| Couch stretch (hip flexor) | 1×60s per side | Counteracts sitting |
| World's greatest stretch | 1×5 per side | Full body mobility |

**Total time:** 15–20 min. Easy enough to do during recovery without affecting next quality.

## When to Modify

| Signal | Action |
|---|---|
| Shin/calf soreness >24h after Session A leg block | Drop weight, hold 5s eccentric tempo, slower progression |
| GCT drift >8% on easy runs (Decision Matrix #5) | **Don't** add strength — investigate run volume/shoes/recovery first |
| Race week (taper) | Skip Session A leg block, keep upper at 50%. Skip Session B entirely. |
| Recovery week | Session A at 70% volume. Session B optional/skip. |
| LTHR/FTP test week (W23/W24) | Skip Session B the day before test. Session A normal Mon. |
| Coming back from illness | Hold strength 2 weeks while volume rebuilds. |

---

## What NOT to do

- ❌ Heavy back squats or deadlifts > 1.5× BW — too much CNS load for our setup
- ❌ Plyometrics in same week as Tempo + Long Run — overload
- ❌ Adding a 3rd day "to catch up" — recovery is when adaptation happens
- ❌ Lift to failure on leg block — these are prevention exercises
- ❌ Skipping Mon "because tired" — that's when injury risk is highest, do reduced version

---

## Why this structure works for HM goal

1. **Single mandatory session** = realistic adherence (>90% expected vs ~60% for 2-mandatory plans).
2. **Stacked on Mon cycling day** = all the load lives together, far from running quality.
3. **Legs on cycling day** = zero impact on Wed Long Run and Sat Tempo.
4. **Optional Thu session** = mobility upkeep without injury risk to weekly quality.
5. **Eccentric calf protocol (Alfredson)** = strongest evidence base for shin/calf prevention in runners.
6. **Trainer → eat → gym matches existing routine** = no new habits to build.
